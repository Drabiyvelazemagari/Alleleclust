AlleleClust is a Python-based tool for allele-level clustering of coding sequences (CDS) based on exact nucleotide sequence identity. The software was developed to support comparative genomic analyses of antimicrobial resistance genes and other conserved loci by providing a reproducible framework for defining and organizing allele groups across large genomic datasets. AlleleClust operates directly on CDS FASTA files generated by the BAKO pipeline and integrates metadata from the corresponding BAKO SUMMARY table to annotate allele clusters with taxonomic and replicon information. Using deterministic exact-sequence clustering, the program groups CDS sequences into allele sets, assigns stable allele identifiers, and records cluster membership together with associated genomic context such as genus, species, and replicon type (chromosome, plasmid, or phage).

The software generates standardized tabular outputs describing allele membership, allele characteristics, and representative sequence selection, facilitating downstream analyses including allele frequency studies, comparative genomics, evolutionary analysis, and host-distribution mapping of resistance determinants. In addition to tabular outputs, AlleleClust produces FASTA datasets containing allele-tagged sequences and representative sequences for each allele cluster, allowing direct integration with phylogenetic, alignment, or structural analysis workflows. Optional outputs include FASTA files for individual allele groups and representative subsets.

AlleleClust supports both command-line and graphical user interface (GUI) execution and is implemented as a single portable Python script requiring minimal dependencies. The tool is designed to integrate seamlessly into genomic analysis pipelines and provides a transparent and reproducible approach for allele definition and sequence clustering in bacterial genomic datasets.
